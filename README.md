# EduTask Manager

Простое учебное консольное приложение для управления задачами. Позволяет добавлять, просматривать и удалять задачи. Проект предназначен для студентов, изучающих основы разработки, структуру репозитория и оформление технической документации.

---

## Описание проекта

EduTask Manager — это лёгкое консольное приложение, созданное для демонстрации базовых принципов программирования и работы с Git/GitHub. Проект подходит для начинающих разработчиков.

Основные функции:

* Добавление новой задачи
* Просмотр списка задач
* Удаление задач
* Сохранение данных в локальном файле

---

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/username/edutask-manager.git
cd edutask-manager
```

### 2. Проверка версии Python

```bash
python --version
```

### 3. Запуск приложения

```bash
python main.py
```

---

## Примеры использования

### Добавление задачи

```
Команда (add/list/remove/exit): add
Введите текст задачи: Подготовить README.md
Задача добавлена!
```

### Просмотр задач

```
Команда (add/list/remove/exit): list
1. Подготовить README.md
```

### Удаление задачи

```
Команда (add/list/remove/exit): remove
Введите номер задачи: 1
Задача удалена!
```
