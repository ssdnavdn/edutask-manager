EduTask Manager

Простое учебное приложение для управления задачами. Позволяет создавать, редактировать, удалять и просматривать список задач. Проект предназначен для студентов, изучающих основы разработки и работу с Git/GitHub.

Описание проекта

EduTask Manager — это консольное приложение, демонстрирующее базовые принципы программирования, работы со структурой проекта и оформления технической документации.

Основные возможности:

Добавление новой задачи

Просмотр всех задач

Удаление задач

Сохранение данных в локальный файл

Минимальный интерфейс для учебных целей

Программа подходит для начинающих разработчиков, изучающих Python и управление версиями.

Установка и запуск
1. Клонирование репозитория
git clone https://github.com/username/edutask-manager.git
cd edutask-manager

2. Установка зависимостей

Приложение использует стандартную библиотеку Python, поэтому внешние зависимости отсутствуют.

Проверь, что у тебя установлен Python 3.10+:

python --version

3. Запуск программы
python main.py

Примеры использования
▶ Добавление задачи
Введите команду: add
Введите текст задачи: Выучить основы Git
Задача добавлена!

▶ Просмотр задач
Введите команду: list
1. Выучить основы Git
2. Подготовить README.md

▶ Удаление задачи
Введите команду: remove
Введите номер задачи: 1
Задача удалена!
